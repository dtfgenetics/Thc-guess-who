# Who Took It?

**Who Took It?** is a THC — Teaching Healthy Cultivation mystery deduction game.

Players ask yes/no questions, eliminate suspects, and solve the hidden mystery:

> **[Suspect] took [Item].**

Core missing items:

- Bag
- Dabs
- Lighter
- Chocolate Bar
- Gummies

This repository is the project home for the physical printable prototype, game master documents, structured data, and future digital version.

## Current Version

**Prototype:** v0.8  
**Status:** Printable low-ink playtest kit created. Final character art should wait until playtesting confirms the rules, clue balance, and mode structure.

## Supported Modes

- 2-player Duel Mode
- 3+ player Shared Mystery Mode
- Team Mode
- Single-player digital mode planned

## Core Rule

Every clue and question must resolve to a clear **yes** or **no**. No maybe answers, no subjective guesses, and no hidden interpretation.
