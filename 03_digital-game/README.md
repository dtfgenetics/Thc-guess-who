# Digital Game

Future browser/mobile version of **Who Took It?**.

Build order:

1. Load suspects, items, and question bank from `02_data/`.
2. Randomly select a hidden suspect and hidden item.
3. Display a 5x5 board.
4. Use preset question buttons only for v1.
5. Answer questions from binary traits.
6. Allow the player to eliminate suspects and make an accusation.

Do not add free-form typed questions until the binary rules engine is tested.
