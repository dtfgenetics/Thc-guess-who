# Printable Prototype v0.8

Low-ink printable playtest kit for **Who Took It?**.

Includes locally generated files:

- player boards
- suspect cards
- item cards
- mystery cards
- rules sheet
- question helper
- elimination sheets
- playtest notes
- designer balance sheet
- complete ZIP kit
- complete combined PDF

This is a prototype for testing gameplay, not final art.

## Print Notes

Test one card sheet before printing the full deck double-sided. Printer alignment can vary.
